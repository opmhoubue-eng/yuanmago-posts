开发与运行环境

为保证源码能够顺利运行，请按照以下环境要求搭建：

  1. 服务器环境：IIS + PHP 支持

  3. 开发工具：VS2015（管理后台需 VS2017 或更高版本）
  4. Cocos Dashboard：安装 Creator 2.4.3
