新品FXcoin交易所源码 | 杠杆合约+控盘分销 | FastAdmin框架二开版本

这套 FXcoin交易所源码 为最新版本，基于 FastAdmin框架二次开发，支持 币币交易、法币交易、杠杆合约、控盘模式、代理分销体系，并附带 原生APP源码，功能完整，适合研究和二次开发。

核心特点：

  • FastAdmin框架二开：前后台逻辑清晰，扩展性强。
  • 多种交易模式：支持币币、法币、杠杆合约，满足多样化需求。
  • 控盘分销机制：支持多级分销与后台控盘功能。
  • 原生APP源码：安卓前端源码，已实测可用，UI流畅。
  • 国际化适配：适合海外运营的数字资产交易平台。

测试环境

  • 服务器：CentOS 7.6
  • 管理工具：宝塔面板
  • Web服务：Nginx + PHP7.3
  • 数据库：MySQL 5.6
  • 进程管理：Supervisor 2.2

安装部署步骤

  1. 更新服务器

    yum update

  2. 安装BT集成环境

  3. 安装依赖

    • Nginx
    • MySQL 5.6
    • PHP 7.3
    • Supervisor 2.2

  4. 配置PHP

    • 安装扩展：redis、Swoole4
    • 删除禁用函数：shell_exec、proc_open

  5. 修改配置文件

    • /application/config.php：代币服务外网IP
    • /application/extra/swoole.php：外网IP
    • database.php：数据库配置

    • 调试模式：

      'app_debug' => true, 'app_trace' => true,

  6. 放通端口：3306、6379、9111、8069

  7. 上传配置文件

    • 上传并解压 profile.zip 到 /www/server/panel/plugin/supervisor/profile

  8. 导入数据库

    • 导入 ex2.zip，并按要求清空/保留指定表数据。

  9. 测试运行

    • 生成模拟账号，检查钱包地址是否正常生成。
  10. 后台登录
  • 地址：http://域名/wQ2pdlSeK4.php
  • 账号：admin
  1. APP打包
  • 支持原生APP打包（安卓），可根据需求扩展。
